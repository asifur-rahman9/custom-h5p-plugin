## 📖 Description
Lightweight WordPress/Pressbooks plugin that applies custom CSS styles to H5P content, enhancing interactive activities.

### Features
- **Global Font** – Montserrat applied to all H5P modules  
- **Accordion Styling** – custom colors, icons, hover effects  
- **Question Sets** – themed backgrounds, buttons, feedback  
- **Button UI** – unified styling for H5P interface buttons  
- **Control Hiding** – optionally hide progress indicators (dots)

## 🚀 Installation
```bash
git clone https://github.com/YourUsername/custom-h5p-styles.git
# copy `custom-h5p-styles` to /wp-content/plugins/
# then activate in Plugins → Installed Plugins

🛠️ Usage
	1.	Appearance → Customize → Additional CSS
	2.	Paste contents of h5p-styles.css, Publish.
	3.	Edit CSS there to tweak fonts/colors; changes apply instantly.
